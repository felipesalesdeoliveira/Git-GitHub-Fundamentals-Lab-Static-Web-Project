Início

Esta é a base inicial do laboratório. Use este arquivo para simular alterações em múltiplas branches e praticar merge/rebase.

Objetivos

  • Praticar versionamento com Git
  • Criar e revisar Pull Requests
  • Resolver conflitos manualmente
  • Versionar releases com tags

Fluxo de Trabalho

  1. Criar branch a partir de develop
  2. Implementar uma feature
  3. Commitar com mensagem semântica
  4. Abrir Pull Request
  5. Fazer merge após revisão
